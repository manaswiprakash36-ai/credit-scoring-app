# Credit Risk Prediction System

A Machine Learning based web application that predicts whether a customer is financially safe or risky using credit history data.

## Features
- Credit risk prediction
- Interactive frontend UI
- Random Forest ML model
- Live deployed Streamlit app

## Technologies Used
- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy

## Project Files
- app.py → Frontend + prediction logic
- credit_model_small.pkl → Trained ML model
- requirements.txt → Required libraries
